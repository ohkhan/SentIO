How it Works

Blockchain as a Service (BAAS)

Serves Blockchain specific functionality through containerized web service wrapped around the Blockchain.

Key Store

Asymmetric key pairing. User's public keys are stored in a separate container but used to sign all requests pushed to Blockchain, tying the user to their data.

Ledger Update

Maintains a ledger based on the successful completion of Blockchain requests. Requiring atomicity with the guarantee of consistency.
